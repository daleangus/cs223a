This application uses your implementation of dh.py to visualize the forward kinematics of a robot.

Each row of dh_table specifies a joint frame of the robot. For prismatic joints, keep di empty, and for revolute joints, keep θi empty. These joint variables are specified by q.

To add a new link to the robot, fill in the last row of dh_table. A new blank row will appear to continue adding new links.

Key:

  • T_ee_to_0 - transformation matrix from the end-effector to the base
  • dh_table - input (ai-1, αi-1, di, θi) DH parameters
  • q - input desired joint positions
