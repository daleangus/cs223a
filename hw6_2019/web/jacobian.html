This application uses your implementation of jacobian.py to visualize the linear and angular Jacobians. The Jacobians here are expressed in frame {0}.

Each column j of the Jacobian corresponds to the velocity at the end-effector produced by moving joint j. Click on a joint to see its linear and angular velocity contributions at the end-effector. Notice how the directions of these vectors are equal to the corresponding columns in the Jaocbians.

Each row i of the Jacobian corresponds to the joint torques required to produce the Cartesian force or moment at the end-effector. Click on a direction in the end-effector frame to see the torques required to produce a force or moment in that direction. Notice how the magnitudes of these torques are equal to the corresponding rows in the Jacobian.

You can modify the given robot by changing the dh_table parameters or moving the end-effector position to a different ee_offset. See the DH Parameters visualization for more details on how to modify the DH parameters.

Key:

  • J_v - linear Jacobian in frame 0
  • J_w - angular Jacobian in frame 0
  • dh_table - input (ai-1, αi-1, di, θi) DH parameters
  • ee_offset - input end-effector offset in the last frame
  • q - input desired joint positions
